# Blog Personal

Aplicación web tipo blog donde se muestran posts dinámicos con información, imágenes y un sistema de lectura completa mediante modales.

## Funcionalidades

- Renderizado dinámico de posts desde JavaScript
- Vista previa de cada post en formato card
- Botón **“Leer más”** que abre un modal con el contenido completo
- Uso de imágenes locales para cada post
- Diseño responsive usando Bootstrap
- Código organizado y escalable

## Tecnologías

- HTML
- CSS
- JavaScript (ES6+)
- Bootstrap 5

## Estructura del proyecto

- Renderizado dinámico con `innerHTML`
- Manejo de eventos mediante delegación de eventos
- Uso de atributos `data-*` para identificar posts
- Modal reutilizable para mostrar contenido completo

## Aprendizajes

- Manipulación del DOM de forma dinámica
- Buenas prácticas en manejo de eventos (sin `onclick` en HTML)
- Uso de modales para mostrar información extendida
- Organización de contenido en estructuras de datos (arrays de objetos)
- Manejo correcto de rutas relativas para assets
